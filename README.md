# 大学生课表学习计划生成器

一个本地运行的网页应用：录入大学课程表、课程难度和近期任务后，系统先用规则生成每周学习计划，再在配置 OpenAI API Key 时进行中文润色和个性化优化。

## 功能

- 表单录入学生信息、学期目标、每天可用自习时间和课程条目。
- 每门课支持填写星期、上课时间、地点、课程类型、难度、近期任务/考试。
- 自动生成周一到周日计划，包含课前预习、课后复盘、作业项目、考试复习和缓冲整理。
- 没有 OpenAI API Key 时仍可使用规则生成的基础计划。

## 运行

1. 安装依赖：

```bash
npm install
```

2. 配置环境变量：

```bash
cp .env.example .env
```

将 `.env` 里的 `OPENAI_API_KEY` 替换为你的 OpenAI API Key。如果不配置，应用会自动返回规则生成的基础计划。

3. 启动开发服务：

```bash
npm run dev
```

前端默认运行在 `http://localhost:5173`，后端默认运行在 `http://localhost:3001`。

## 验证

```bash
npm test
```
